# Team Task Manager

A full-stack MERN application where users can create projects, assign tasks, manage teams, and track task progress with role-based authentication.

## Features

- User Authentication (JWT)
- Role-Based Access Control (Admin / Member)
- Create and Manage Projects
- Create, Assign and Update Tasks
- Dashboard Statistics
- MongoDB Database Integration
- REST API Architecture
- Responsive Frontend UI

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs
